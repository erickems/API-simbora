# API-simbora

1. Abra o terminal dentro da pasta principal do projeto e rode os comandos
```
$ npm install
$ npm start
```

2. Abra o postman e digite
```
$ http://localhost:8080/<rota>
```

3. As rotas:
```
$ {GET} =  /clientes
$ {GET} =  /estabelecimentos

$ {POST} =  /criaCliente
$ {POST} =  /criaEstabelecimento
```
